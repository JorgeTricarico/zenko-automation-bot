# Brainstorming — Exploring Alternatives

Facilitates ideation sessions before making decisions.

## Format

1. **Define the problem** — what to solve, constraints
2. **Diverge** — minimum 3 alternatives, each with pros/cons/effort/risk
3. **Converge** — evaluate against: feasibility, time, maintainability, impact
4. **Recommend** — preferred option with justification + plan B
